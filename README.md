# Decoding-IPL-A-Comprehensive-Data-Analysis
This project explores IPL data from 2008 to 2024, analyzing team performances, player stats, and match outcomes over 16 seasons. Using EDA, it examines trends in batting, bowling, win rates, and key player contributions to uncover insights that help predict future match results and enhance strategic decision-making.<br/><br/>

Link for dataset used:[https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020/code]
